# Meine Gefühle 🎀

Eine persönliche Web-App im Sanrio-Stil, um Gefühle visuell und ehrlich darzustellen.

## 🚀 Veröffentlichen auf GitHub Pages

### Schritt 1 – Repository erstellen
1. Gehe auf [github.com](https://github.com) und melde dich an.
2. Klicke auf **+** → **New repository**.
3. Gib einen Namen ein (z. B. `meine-gefuehle`) → **Create repository**.

### Schritt 2 – Dateien hochladen
1. Klicke auf **Add file** → **Upload files**.
2. Lade diese drei Dateien hoch:
   - `index.html`
   - `style.css`
   - `script.js`
3. Klicke auf **Commit changes**.

### Schritt 3 – GitHub Pages aktivieren
1. **Settings** → **Pages** (linke Seitenleiste).
2. Branch: `main`, Ordner: `/ (root)` → **Save**.

### Schritt 4 – Link teilen
Nach ca. 1–2 Minuten erreichbar unter:
```
https://DEIN-BENUTZERNAME.github.io/meine-gefuehle/
```
